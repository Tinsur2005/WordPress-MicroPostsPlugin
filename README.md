# WP微语插件

一款轻量化的 WordPress 个人微语/碎碎念发布插件，支持后台独立发布微语，前台专属页面优雅展示所有微语内容。

## ✨ 功能特点

- 📝 **独立微语管理**：后台独立菜单管理微语，支持新增、编辑、删除、批量管理
- 🎨 **优雅展示**：简约清新的卡片式设计，支持深色/浅色主题兼容
- 📱 **响应式适配**：完美适配电脑、平板、手机所有设备
- ⚡ **AJAX分页**：动态加载分页，无需刷新页面
- 📊 **浏览统计**：实时统计页面浏览人数
- 🔒 **安全可靠**：严格遵循 WordPress 安全开发规范
- 📁 **独立数据**：使用独立自定义数据表，不影响 WordPress 核心数据

## 🚀 安装步骤

### 方法一：手动安装

1. 下载插件压缩包
2. 登录 WordPress 后台 → **插件** → **安装插件** → **上传插件**
3. 选择下载的压缩包上传
4. 点击 **启用插件**

### 方法二：FTP安装

1. 将 `wp-weiyu-plugin` 文件夹上传到 `/wp-content/plugins/` 目录
2. 登录 WordPress 后台 → **插件**
3. 找到「WP微语插件」并点击 **启用**

## 📖 使用方法

### 发布微语

1. 后台左侧菜单点击 **微语管理**
2. 点击 **发布新微语** 按钮
3. 输入微语内容（支持文字、表情、换行）
4. 设置发布时间（可选）
5. 点击 **发布微语**

### 创建前台页面

1. 后台 → **页面** → **新建页面**
2. 输入页面标题（如「微语」）
3. 在右侧「页面属性」→「模板」中选择「微语页面」
4. 点击 **发布**
5. 访问该页面即可查看所有微语

### 自定义设置

1. 后台 → **微语管理** → **设置**
2. 可自定义页面标题和副标题

## 🔒 安全特性

- ✅ SQL注入防护：所有数据库操作使用参数化查询
- ✅ CSRF防护：所有表单提交有 nonce 验证
- ✅ XSS防护：使用 `esc_html()`、`esc_attr()` 等输出转义
- ✅ 权限控制：仅管理员可发布、管理微语
- ✅ 输入过滤：使用 `sanitize_textarea_field()` 过滤内容

## 📊 数据库结构

插件使用独立自定义数据表 `wp_weiyu_posts`：

| 字段 | 类型 | 说明 |
|------|------|------|
| id | mediumint(9) | 主键，自增 |
| content | text | 微语内容 |
| created_at | datetime | 创建时间 |
| modified_at | datetime | 修改时间 |
